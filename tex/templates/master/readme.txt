Для корректной работы шаблона необходимо:
1. Вручную установить пакет pscyr.
Инструкция по установке: http://blog.harrix.org/?p=444
2. Для версии MikTeX x64 вручную скачать Biber.exe.
3. Сборку производить с использованием Biber.
В TeXstudio это можно сделать следующим образом:
Параметры -> конфигурация TeXstudio -> Построение -> Библиография по умолчанию -> Biber
4. Для редактирования литературы следует воспользоваться программой JabRef или другой аналогичной.
http://jabref.sourceforge.net
5. Для сборки библиографии нужно воспользоваться командой
Инструменты -> Библиография (F11)
или следующей последовательностью сборки в 4 этапа:
pdflatex
biber
pdflatex
pdflatex
